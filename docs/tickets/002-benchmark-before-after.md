# Benchmark the before-and-after workflow

Status: Open

## Goal

Produce an honest comparison showing how often Let Him Grill avoids unnecessary
interruptions without crossing decisions that require human judgment.

## Scenarios

- Define 5–10 representative planning tasks with several dependent decisions.
- Run each task once with a conventional question-by-question workflow and once
  with Let Him Grill.
- Use the same task, repository context, and success criteria for both runs.
- Keep the raw prompts and results so published claims can be checked.

## Measurements

Record for each run:

- individual questions asked
- decisions resolved autonomously
- human gates reached
- time until a usable plan
- dependent decisions reassessed after an earlier choice changes

Report aggregate results without hiding unfavorable scenarios. Treat
`8 questions → 2 human gates` as an example until the data supports it.

## Acceptance criteria

- At least five comparable scenarios are recorded.
- Raw results and the aggregation method are available in the repository.
- The README contains one concise, evidence-backed before-and-after result.
- No claim is based on an invented or selectively chosen run.
